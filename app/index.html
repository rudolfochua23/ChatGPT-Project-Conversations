Setting Up Your Account

1

Open the App

Visit the ChatPileAI URL (e.g. https://chatpileai.yourdomain.com). You'll see this landing page.

2

Create an Account

Click "Get Started" or "Create account". Fill in your email, choose a username (3-30 chars), and a password (min 12 chars).

3

Verify Your Email

Check your inbox for a 6-digit verification code. Enter it on the verification screen. The code expires in 15 minutes.

4

Start Browsing

After verification you're automatically logged in. Browse conversations in the left panel, read messages in the center, and view code snippets on the right.

5

Forgot Your Password?

Click "Forgot password?" on the sign-in screen. Enter your email and check your inbox for a reset link. The link expires in 1 hour.

Setting Up Tampermonkey

1

Install Tampermonkey Extension

Install Tampermonkey for your browser: tampermonkey.net. Available for Chrome, Firefox, Edge, and Safari.

2

Get Your API Key

Sign in to ChatPileAI → click Settings → copy your Tampermonkey API Key. This key links saved conversations to your account.

3

Add the Script

Open the Tampermonkey dashboard → click "Create a new script" → delete the default template → paste the full script from tampermonkey-script.example.md → update the API_URL with your dashboard URL → save (Ctrl+S).

4

Set Your API Key

Visit any supported AI chat site. You'll see the ChatPileAI panel in the corner. Click "Set API Key" and paste the key you copied from Settings.

5

Start Chatting

Conversations auto-save every minute. Click "Save" for manual saves. Saved chats appear instantly in your dashboard — no rebuild or deploy needed.

Tampermonkey Script

Get the full script from the tampermonkey-script.example.md file in the project repository. Below is the key configuration section:
